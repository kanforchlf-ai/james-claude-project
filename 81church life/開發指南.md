> 想改網站功能時看這份。「想改 X → 改哪個檔案 → 怎麼推上線」一頁式速查。

## 想改什麼 → 改哪 → 怎麼推

| 想改的東西 | 改哪個檔案 | 怎麼推 |
| --- | --- | --- |
| 出席計算邏輯（活動權重、基數、挽回判定…） | `update_dashboard.py` | A. 走 `deploy.sh` |
| 自動生成頁面的內容/版面（見下方清單） | `update_dashboard.py` 裡對應 function | A. 走 `deploy.sh` |
| 共用 CSS / 主題色 | `81Y3-dashboard/theme.css` | B. 內層 push |
| 手寫頁面內容/版面（見下方清單） | `81Y3-dashboard/<檔案>.html` 直接改 | B. 內層 push |
| LINE Bot 查詢功能 | `81Y3-dashboard/bot_server.py` 或 `dashboard_rag.py` | B. 內層 push（Render 自動重部署） |
| 從 Google Sheet 同步名單 | `sync_from_gsheet.py` | 外層 commit + push |

## ⚠️ 自動生成 vs 手寫頁面

### 自動生成（**改了會被覆蓋，不要直接動**）

`update_dashboard.py` 每跑一次就**完全覆蓋**這些檔案：

```
index.html              ← 首頁
weekly.html             ← 本週點名
trend.html              ← 出席趨勢
cowork.html             ← 配搭出席總覽
kids/index.html         ← 兒童專區

y1, y2, y3/             各區 index.html、invite.html、leaderboard.html
hs1, hs2, hs3/          各區 index.html、invite.html、leaderboard.html
ms1, ms2/               各區 index.html、invite.html、leaderboard.html
```

要改這些頁面的版面/內容 → **去 `update_dashboard.py` 改產生 HTML 的程式碼**，不要改 HTML 本身。

### 手寫（**可以直接改**）

```
81Y3-dashboard/
├── youth.html      ← 青年群組頁
├── hs.html         ← 高中群組頁
├── ms.html         ← 國中群組頁
├── guide.html      ← 系統說明
├── leaderboard.html ← 全會所排行榜
└── theme.css       ← 配色 / 樣式
```

LINE Bot 相關也是手寫：`bot_server.py`、`dashboard_rag.py`、`Dockerfile`、`requirements.txt`、`LINE_BOT_README.md`。

## 兩種 push 流程

### A. 走 deploy.sh（資料/邏輯/自動生成頁面）

```bash
cd "/c/Users/james/OneDrive/Desktop/Claude-workspace/projects/81church life"
./deploy.sh
```

deploy.sh 自動：跑 `update_dashboard.py` → 內層 push → 外層 push。

**⚠️ 一律在主目錄跑，不要在 worktree**（worktree 沒有內層 `.git`，會推到錯的地方）。

### B. 內層直接 push（靜態檔案 / LINE Bot）

```bash
cd "/c/Users/james/OneDrive/Desktop/Claude-workspace/projects/81church life/81Y3-dashboard"
git add <改的檔案>
git commit -m "..."
git push
```

→ 推到 `kanforchlf-ai/81Y3-dashboard`，GitHub Pages 1-2 分鐘後上線。

## Repo 職責切分

| Repo | 內容 | 用途 |
| --- | --- | --- |
| **內層** `kanforchlf-ai/81Y3-dashboard` | 整個 dashboard 網站靜態檔案 + LINE Bot | GitHub Pages 部署 + Render LINE Bot |
| **外層** `kanforchlf-ai/james-claude-project` | `update_dashboard.py`、`deploy.sh`、`sync_from_gsheet.py`、雜項腳本、README、本指南 | 工作流 source code 備份 |

外層**不再追蹤** `81Y3-dashboard/` 子資料夾（`.gitignore` 處理）。所以：

- 改 `81Y3-dashboard/` 裡的東西 → 只在內層 commit
- 改 `81Y3-dashboard/` 外的東西（`update_dashboard.py` 等）→ 在外層 commit
