# 政治獻金與財產申報分析 — 專題計畫

*最後更新：2026-04-13*

---

## 專題目標

透過政府公開資料（政治獻金、財產申報、選舉資料），系統性揭露台灣政治人物的金流結構與潛在弊端，包含：跨候選人捐款連結、家族政治網絡、資產異常變動等。

---

## 頁面方向：兩條路線

### 公司既有頁面（election.html）
**網址**：https://jameskan-tw.github.io/james-claude-project/political-donation/政治人物選舉紀錄.html

## 資料來源

| 檔案 | 內容 | 格式 |
|------|------|------|
| `parties.csv` | 中選會選舉候選人資料（參選紀錄、政黨、當選/落選） | CSV |
| `supplement.csv` | 補充資料（不分區立委等 parties.csv 缺漏項目） | CSV |
| `incomes.csv` | 政治獻金收入明細（793,343 筆） | CSV |
| `list.csv` | 廉政專刊財產申報名單 | CSV |
| `spouse.csv` | 配偶姓名對照（從 基本資料.sqlite 匯出，3,445 筆） | CSV |
| `ccm_search.csv` | 縣市議員親屬關係對照（readr-media，979 筆） | CSV |
| `政治獻金.sqlite` | 政治獻金資料庫 | SQLite |
| `基本資料.sqlite` | 財產申報基本資料（含配偶欄位，子女全部匿名） | SQLite |
| `scoring_rules.json` | 財產異動評分規則 | JSON |

**資料限制備注**
- `parties.csv` 涵蓋範圍約自 1994 年後，更早的政治人物（如賴清海）無選舉紀錄
- `ccm_search.csv` 以縣市議員為主，立委覆蓋率低
- `基本資料.sqlite` 子女姓名全部以 ○ 匿名，無法做親子連結
- 親屬偵測：同選區同姓演算法誤報率過高，已放棄；改以 `ccm_search.csv` 人工核對資料為準

## 已完成的分析模組

### 1. 跨候選人共同捐款人分析（`analyze.js`）
- **輸出**：`report.html`
- 找出同一批人同時捐款給不同立委候選人的群體（≥10 位共同捐款人）

### 2. 接棒型政治世家偵測（`dynasty.js`）
- **輸出**：`dynasty_report.html`
- 同一選區、同姓氏、不同人，接棒間隔 ≤4 年；共找到 621 組
- 演算法局限：同名同姓多，誤報率高，僅供參考，不整合進 profile

### 3. 個人政治履歷產生器（`profile.js`）
- **輸出**：`{姓名}_政治履歷.html`
- 選舉紀錄、政黨歷程、職位時間軸、**親屬關係（ccm_search）**、配偶政治背景

### 4. 多人整合履歷瀏覽器（`combined_profile.js`）
- **輸出**：`政治人物選舉紀錄.html`（GitHub Pages 公開）
- 左側欄人物列表 + 右側面板；含親屬關係段落（ccm_search）
- 示範人物：蔡英文、韓國瑜、柯文哲、柯建銘、苗博雅、**蕭景田**（親屬範本）

### 5. 家族金流分析（`family_donation.py`）
- **輸出**：`{姓名}_家族金流.html`
- 同時捐款給同姓多位候選人的捐款人；同縣市篩選避免同黨誤報

### 6. 家族網絡視覺化原型（`family_network.js`）⬅ 新
- **輸出**：`賴清海_家族網絡.html`
- CSS 卡片樹狀圖：直系連線（兒子）＋橫向連線（姻親）
- 每卡顯示黨籍、選區、當選次數；底部附選舉明細表
- **Route B 概念原型**，尚未整合捐款資料

### 7. 財產申報異動分析（`export_sqlite.py` + 評分規則）
- **輸出**：`財產申報異動分析表.html`

### 8. 黃國昌專項分析
- `黃國昌時間軸.html`、`黃國昌時間軸圖.html`、`黃國昌關係地圖.html`

## 技術架構

```
CSV / SQLite 原始資料
        ↓
  Node.js / Python 分析腳本
        ↓
  HTML 報告（瀏覽器直接閱讀）
        ↓
  GitHub Pages 公開
```

- **腳本語言**：Node.js（選舉/捐款分析）、Python（財產申報匯出）
- **輸出格式**：HTML（Noto Sans TC，白底深字）
- **部署**：GitHub Pages（`jameskan-TW/james-claude-project`）
- **公開網址**：`https://jameskan-tw.github.io/james-claude-project/political-donation/`
