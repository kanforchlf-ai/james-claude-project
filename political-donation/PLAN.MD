# 政治獻金與財產申報分析 — 專題計畫

## 專題目標

透過政府公開資料（政治獻金、財產申報、選舉資料），系統性揭露台灣政治人物的金流結構與潛在弊端，包含：跨候選人捐款連結、家族政治網絡、資產異常變動等。

---

| 檔案 | 內容 | 格式 |
|------|------|------|
| `parties.csv` | 中選會選舉候選人資料（參選紀錄、政黨、當選/落選） | CSV |

## 已完成的分析模組

### 1. 跨候選人共同捐款人分析（`analyze.js`）
- **輸出**：`report.html`
- **邏輯**：找出同一批人同時捐款給不同立委候選人的群體，揭示候選人之間的利益連結
- **門檻**：≥10 位共同捐款人才納入

### 2. 接棒型政治世家偵測（`dynasty.js`）
- **輸出**：`dynasty_report.html`
- **邏輯**：同一選區、同姓氏、不同人，接棒間隔 ≤4 年
- **結果**：共找到 621 組疑似世家

### 3. 個人政治履歷產生器（`profile.js`）
- **輸出**：`{姓名}_政治履歷.html`
- **資料**：選舉紀錄、政黨歷程、職位時間軸、配偶政治背景
- **已產出人物**：黃國昌、傅崐萁、吳沛憶、徐巧芯、蘇巧慧、蔡英文、韓國瑜、柯文哲、苗博雅

### 3b. 多人整合履歷瀏覽器（`combined_profile.js`）
- **輸出**：`政治人物選舉紀錄.html`
- **設計**：左側欄人物列表 + 右側面板，點選即切換，單一 HTML 無需伺服器
- **內容**：人物基本資料、4 格統計卡、政黨歷程、參選紀錄表、廉政專刊職位時間軸、政治獻金收入長條圖
- **示範人物**：蔡英文、韓國瑜、柯文哲、柯建銘、苗博雅

### 4. 家族金流分析（`family_donation.py`）
- **輸出**：`{姓名}_家族金流.html`
- **邏輯**：找出同時捐款給同姓多位候選人的捐款人，揭示家族政治網絡
- **已產出人物**：黃國昌、傅崐萁、吳崢、徐榛蔚、蔡其昌、蘇巧慧、賴品妤

### 5. 財產申報異動分析（`export_sqlite.py` + 評分規則）
- **輸出**：`財產申報異動分析表.html`
- **邏輯**：對比各期申報資料，標記異常增減，依 `scoring_rules.json` 評分

### 6. 黃國昌專項分析
- `黃國昌時間軸.html` — 政治事件時間軸（文字版）
- `黃國昌時間軸圖.html` — 視覺化時間軸
- `黃國昌關係地圖.html` — 捐款人關係網絡圖

## 分析缺口 / 待補項目

| 優先 | 項目 | 說明 |
|------|------|------|
| 高 | 整合履歷擴充人物 | 將吳崢、蔡其昌、徐榛蔚、賴品妤等補入 `政治人物選舉紀錄.html` |
| 高 | 捐款人身份查核 | 揭露主要大額捐款人的職業、公司、與政治人物的潛在利益關係 |
| 中 | 得票數資料補充 | parties.csv 無得票欄，需補入才能做選舉比較長條圖 |
| 中 | 捐款→標案交叉比對 | 比對捐款人的公司是否有承接政府標案 |
| 中 | 全候選人財產異動排名 | 哪些人資產增幅最異常（依評分規則排名） |
| 低 | 跨黨捐款人分析 | 同一捐款人跨黨捐款行為的動機分析 |
| 低 | 地理視覺化 | 捐款人、候選人的地理分布地圖 |

## 技術架構

```
CSV / SQLite 原始資料
        ↓
  Node.js / Python 分析腳本
        ↓
  HTML 報告（瀏覽器直接閱讀）
```

- **腳本語言**：Node.js（選舉/捐款分析）、Python（財產申報分析）
- **輸出格式**：HTML（風格：白底深字、Noto Sans TC、max-width 720px）
- **資料庫**：SQLite（本地，不需伺服器）
